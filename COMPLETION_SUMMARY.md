## 📊 技术实现要点

### 照片管理功能
- 调用 Coze API 的 search 接口获取所有照片
- 实现前端搜索和筛选逻辑
- 动态生成标签筛选器

### 使用统计功能

- 标签分布统计和排序
- 色彩风格分布统计

### 数据导出功能
- 使用 Blob API 生成 JSON 文件
- 使用 URL.createObjectURL 创建下载链接
- 自动触发浏览器下载
